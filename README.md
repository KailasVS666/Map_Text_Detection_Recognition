# 🗺️ Rumsey Map OCR — Historical Map Text Recognition

An end-to-end deep learning pipeline for detecting and recognizing text in
high-resolution historical maps from the David Rumsey Map Collection.

**Dataset:** ICDAR 2024 Historical Map Text Challenge  
**Approach:** Fine-tuned PaddleOCR (PP-OCRv4) on 34,521 ground-truth word annotations  
**Status:** Active Development — Fine-tuning Phase

---

## 📊 Dataset

| Split | Maps | Word Annotations |
|-------|------|-----------------|
| Train | 200  | 34,521          |
| Val   | 40   | 5,543           |

Ground truth annotations include: polygon vertices, transcribed text, illegibility flags, truncation flags.

## 🚀 Quick Start

### Step 1: Extract Word Crops from Ground Truth
```bash
python step1_extract_icdar_crops.py
```
Extracts ~34,521 labeled word crops from ICDAR annotations. Takes ~10-15 min.

### Step 2: Download Pretrained Models
```bash
mkdir pretrained && cd pretrained

# Recognition model
curl -O https://paddleocr.bj.bcebos.com/PP-OCRv4/english/en_PP-OCRv4_rec_train.tar
tar -xf en_PP-OCRv4_rec_train.tar

# Detection model
curl -O https://paddleocr.bj.bcebos.com/PP-OCRv4/english/en_PP-OCRv4_det_train.tar
tar -xf en_PP-OCRv4_det_train.tar
```

### Step 3: Fine-tune Recognition Model
```bash
python step2_finetune_recognition.py
```
Fine-tunes PP-OCRv4 on historical map fonts. ~100 epochs, ~2-4 hours on GPU.

### Step 4: Fine-tune Detection Model
```bash
python step3_finetune_detection.py
```
Fine-tunes DBNet to detect text in maps (reduces false positives on hachures/symbols).

### Step 5: Evaluate & Run Inference
```bash
python step4_evaluate_and_infer.py
```

## 🎯 Optimization Strategy

```
Phase 1: ICDAR Ground Truth Fine-tuning (current)
  ├── 34,521 human-labeled word crops → recognition model
  └── Polygon annotations → detection model

Phase 2: Hard Negative Mining (after phase 1)
  ├── Mine remaining errors (50-85% confidence)
  ├── Filter: keep real text, delete map symbols
  └── Add to training set → retrain

Phase 3: Post-processing
  ├── Link broken text (multi-word place names)
  ├── Spell-check against historical gazetteer
  └── Export to searchable database
```
